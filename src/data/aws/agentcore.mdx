**Amazon Bedrock AgentCore** is a suite of purpose-built, modular services that handle the production infrastructure for agentic AI applications — hosting, memory, tool integration, authentication, and observability — so you can focus on agent logic instead of undifferentiated infrastructure.

## Runtime — Purpose-Built Agent Hosting

A serverless execution environment designed for agentic workloads. Unlike standard Lambda, it maintains agent context for **sessions up to 8 hours** with **100 MB payload capacity** for multi-modal content.

<div className="flex gap-4 items-start">
  <div className="w-1/2 min-w-0">

    <table className="brand-table">
      <thead><tr><th>Feature</th><th>Detail</th></tr></thead>
      <tbody>
        <tr><td>Isolation</td><td>Each session runs in a dedicated microVM with isolated CPU, memory, and filesystem</td></tr>
        <tr><td>Data safety</td><td>Complete separation prevents cross-session contamination</td></tr>
        <tr><td>Cleanup</td><td>Automatic microVM termination and memory sanitisation after session end</td></tr>

## Memory — Context-Aware Intelligence

Dual memory architecture giving agents the ability to remember and reason over past interactions.

<div className="flex gap-4 items-start">
  <div className="w-1/2 min-w-0">
    <p><strong>Short-term memory</strong></p>
    <ul className="list-arrow">
      <li>Maintains full context within a session</li>
      <li>Automatically scoped to the session ID</li>
      <li>Available for the duration of the microVM lifetime</li>
    </ul>
  </div>
  <div className="w-1/2 min-w-0">
    <p><strong>Long-term memory</strong></p>
    <ul className="list-arrow">
      <li>Persists summaries and key facts across sessions</li>
      <li>Configurable retention (days / session count)</li>
      <li>Enables personalisation and continuity for returning users</li>
    </ul>
  </div>
</div>

## Gateway — Unified Tool Integration

Gateway implements **Model Context Protocol (MCP)** as a standardised way for agents to discover and invoke tools — no custom translation code per tool.

<table className="brand-table">
  <thead><tr><th>MCP Method</th><th>Purpose</th></tr></thead>
  <tbody>
    <tr><td><code>tools/list</code></td><td>List all available tools</td></tr>
    <tr><td><code>tools/call</code></td><td>Invoke a specific tool</td></tr>
    <tr><td><code>x_amz_bedrock_agentcore_search</code></td><td>Semantic tool discovery</td></tr>
  </tbody>
</table>

## Identity — Secure Agent Authentication

Each agent receives a unique identity with fine-grained permissions — similar to Amazon Cognito but purpose-built for agents.
