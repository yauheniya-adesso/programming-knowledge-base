## Extract and Chunk for RAG

The most common pattern: extract text from a document and split it into overlapping chunks for embedding + indexing.

<pre><code className="language-python">{`def extract_and_chunk(file_path: str, chunk_size: int = 400, overlap: int = 50) -> list[dict]:
    """
    Extract text from a PDF and return chunks ready for embedding.
    Each chunk includes the source file name and page number.
    """
    with open(file_path, "rb") as f:
        poller = client.begin_analyze_document(
            model_id="prebuilt-layout",
            body=f,
            content_type="application/pdf",
        )
    result = poller.result()

    chunks = []

    for page in result.pages:
        page_text = " ".join(line.content for line in (page.lines or []))
        words = page_text.split()

        # Sliding window chunking
        i = 0
        while i < len(words):
            chunk_words = words[i : i + chunk_size]
            chunks.append({
                "content": " ".join(chunk_words),
                "source": os.path.basename(file_path),
                "page": page.page_number,
            })
            i += chunk_size - overlap

    return chunks


# Usage
chunks = extract_and_chunk("q3-report.pdf")
print(f"Extracted {len(chunks)} chunks")`}</code></pre>

## Extract Tables as Structured Data

<pre><code className="language-python">{`import pandas as pd

def tables_to_dataframes(result) -> list[pd.DataFrame]:
    """Convert Document Intelligence tables to pandas DataFrames."""
    dfs = []
    for table in (result.tables or []):
        data = {}
        for cell in table.cells:
            col = cell.column_index
            row = cell.row_index
            if col not in data:
                data[col] = {}
            data[col][row] = cell.content

        df = pd.DataFrame(data).sort_index()
        # Promote first row to header
        df.columns = df.iloc[0]
        df = df[1:].reset_index(drop=True)
        dfs.append(df)

    return dfs`}</code></pre>

## Ingestion Pipeline: Blob → Document Intelligence → AI Search

<pre><code className="language-bash">{`Blob Storage (raw-documents/)
    │
    ▼  triggered by Storage Queue message
Document Intelligence (prebuilt-layout)
    │  extract text per page, identify tables
    ▼
Chunking (400 tokens, 50 overlap)
    │
    ▼
Azure OpenAI Embeddings (text-embedding-3-small)
    │
    ▼
Azure AI Search (upload_documents)
    │
    ▼
Agent can now retrieve relevant context via search_documents()`}</code></pre>

<ul className="list-arrow">
  <li>Run this pipeline as an <strong>Azure Function</strong> triggered by a Storage Queue message for fully async, scalable ingestion</li>
  <li>Store the extracted text back to Blob Storage as <code>.txt</code> or <code>.json</code> to avoid re-processing on re-indexing</li>
</ul>
