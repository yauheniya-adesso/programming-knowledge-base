## Pricing Essentials

<div className="rounded-lg border border-blue/40 p-4 mb-2">
  <ul className="list-arrow m-0">
    <li><strong>Messages</strong> — billed per conversation message (classic topics + generative AI replies).</li>
    <li><strong>Sessions</strong> — a session is up to 60 minutes of conversation; billed on a monthly basis.</li>
    <li><strong>Included capacity</strong> — M365 E3/E5 and Power Platform per-user plans include a monthly message allowance.</li>
    <li><strong>Add-on packs</strong> — purchase additional message packs when usage exceeds the included quota.</li>
    <li><strong>Azure OpenAI costs</strong> — generative (AI Search + GPT) answers consume separate Azure token budget.</li>
  </ul>
</div>
