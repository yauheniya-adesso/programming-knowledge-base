CSS (**Cascading Style Sheets**) is used to style and layout web pages,
for example to change colors, fonts, spacing, and positioning.

CSS was first released in **1996**. The current standard, **CSS3**,
represents a collection of modular specifications rather than a single version.

CSS is developed and maintained by the <Icon icon="logos:w3c" height={14} style={{display: 'inline', verticalAlign: 'middle'}} /> **World Wide Web Consortium (W3C)**.
