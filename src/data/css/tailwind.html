Tailwind CSS is a utility-first CSS framework that lets you style elements directly using predefined classes.

Installation (via npm):

npm install -D tailwindcss

Example usage:

<button class="bg-blue-500 text-white px-4 py-2 rounded"> Click me </button>

Common Tailwind CSS utility classes with detailed explanations:

  • Colors – Set background, text, and border colors using a predefined palette.
    bg-blue-500, text-white, text-gray-700, bg-red-200
    Example: bg-blue-500 sets a medium blue background.
    The number (100–900) represents the shade intensity (100 = light, 900 = dark).
    You can also add transparency: bg-blue-500/50 is 50% opacity.
    Full color palette.
  • Spacing – Add margin and padding consistently.
    p-4, px-2, py-3, m-4, mt-2, mb-6
    Example: p-4 adds padding on all sides, m-4 adds margin on all sides.
    Prefixes: t=top, b=bottom, l=left, r=right.
    Axes: x=left+right, y=top+bottom.
    Numbers are based on a scale (e.g., 1=0.25rem, 4=1rem, 16=4rem by default).
  • Typography – Control font size, weight, line height, and alignment.
    text-sm, text-lg, font-bold, text-center
    Example: text-sm = small text, text-lg = larger text.
    Numbers indicate relative sizes; see the Tailwind font-size docs.
    font-bold makes text bold; text-center centers text horizontally.
  • Layout – Manage display, flexbox, grid, width, height, and positioning.
    flex, grid, items-center, justify-between, w-full, h-16
    Example: flex creates a flex container, items-center vertically centers children.
    Width and height classes: w-full = 100% of parent, h-16 = 4rem (based on scale, 1 unit = 0.25rem).
    Use grid classes for layout: grid grid-cols-3 gap-4.
