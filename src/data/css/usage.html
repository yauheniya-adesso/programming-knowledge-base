CSS can be applied to HTML in three main ways:

  • External stylesheet – Recommended approach for maintainability.
    <link rel="stylesheet" href="styles.css">
  • Internal CSS – Defined inside the <head> tag.
    <style> body { color: black; } </style>
  • Inline CSS – Applied directly to an element (use sparingly).
    <p style="color: red;">Text</p>
