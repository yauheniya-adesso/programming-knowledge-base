Cloning and forking GitHub projects:

  • Clone – Create a local copy of a repository from GitHub.
    git clone https://github.com/user/repo.git
  • Fork – Make a personal copy of a repository to modify or extend the project independently.
    Note: origin refers to your fork, and upstream refers to the original repository.
  • View remotes – List configured remote repositories.
    git remote -v
  • Add upstream – Link your fork to the original repository.
    git remote add upstream <clone-URL>
  • Sync fork – Fetch updates from the original repository and merge into your main branch.
    git fetch upstream
git checkout main
git merge upstream/main
    Alternative:
    git checkout main
git pull upstream main
