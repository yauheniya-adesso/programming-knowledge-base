Correcting commit message typos:

  • Check commit history – Find the commit with the typo.
    git log --oneline
  • Amend last commit – Fix the most recent commit message (local commits only).
    git commit --amend -m "Corrected commit message"
  • Edit older commit messages – Reword previous commits interactively (rewrites history).
    git rebase -i HEAD~N
    Replace N with the number of commits to review and mark commits with reword.
  • Revert a commit safely – Undo a commit without changing history (recommended for pushed commits).
    git revert HEAD --no-edit
    Creates a new commit that reverses the changes from the previous commit.
  • Push amended history – Update the remote repository after rewriting commits.
    git push --force
    Use with caution: force-pushing can affect other collaborators.
