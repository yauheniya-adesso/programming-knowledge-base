Correcting commit message typos:

  • Check commit history – Find the commit with the typo.
    git log --oneline
  • Amend last commit – Quickly fix the most recent commit message.
    git commit --amend -m "Corrected commit message"
  • Edit older commit messages – Reword previous commits interactively.
    git rebase -i HEAD~N
    Replace N with the number of commits to review. Mark commits with reword to fix their messages.
  • Push amended commits – If commits were already pushed, use force push carefully.
    git push --force
    Be cautious: this rewrites history on the remote repository and can affect collaborators.
