Using GitHub with SSH:

  • Generate an SSH key – Create a key pair for secure authentication.
    ssh-keygen -t rsa -b 4096 -C "your.email@example.com"
  • Locate the public key – This key will be shared with GitHub.
    ~/.ssh/id_rsa.pub
  • Add the key to GitHub – Copy the public key and add it in GitHub under Settings → SSH and GPG keys.
    cat ~/.ssh/id_rsa.pub
  • Start the SSH agent – Enable SSH key handling in your session.
    eval "$(ssh-agent -s)"
  • Add the key to the agent – Load your private key for authentication.
    ssh-add ~/.ssh/id_rsa
  • Use SSH with GitHub – Clone and interact with repositories using the SSH URL.
    git clone git@github.com:username/repository.git
  • Test the connection – Verify that GitHub accepts your SSH key.
    ssh -T git@github.com
    A successful message confirms SSH authentication is working.
