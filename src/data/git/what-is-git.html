Git is a distributed version control system used to track changes in source code during software development. It allows multiple developers to work on the same project efficiently, safely, and independently.

With Git, every developer has a full copy of the project history on their local machine. This makes Git fast, reliable, and usable even when offline. 

# Initialize a new Git repository
git init

# Check the current status of files
git status

# Add files to staging
git add .

# Commit changes with a message
git commit -m "Initial commit"

Git itself is a command-line tool and works locally. To collaborate with others, Git is commonly used together with remote hosting platforms such as:

  • GitHub – Popular platform for open-source and team collaboration
  • GitLab – Includes CI/CD and self-hosting capabilities
  • Bitbucket – Often used with Atlassian tools like Jira
  • Azure DevOps – Enterprise platform for Git hosting, CI/CD, and project management
