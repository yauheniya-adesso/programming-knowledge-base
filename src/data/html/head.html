Contains metadata about the document, such as the page title, linked CSS files, character encoding, viewport settings for responsive design, author info, and external scripts. Should be placed immediately after the <html> tag. 

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document Title</title>
  <link rel="stylesheet" href="styles.css">
  <meta name="author" content="Yauheniya">
  <meta name="description" content="A sample HTML document for demonstration purposes">
  <script src="script.js"></script>
</head>
