Every Java program runs from the main method inside a class. The file name must match the class name. 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Compile with javac HelloWorld.java, then run using java HelloWorld.

Data Types:

  • Primitive: int, double, boolean, char, byte, short, long, float
  • Reference: Objects, Arrays, Strings
