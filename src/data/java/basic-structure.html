Every Java program runs from the main method inside a class. The file name must match the class name: touch HelloWorld.java. 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Compile with javac HelloWorld.java, then run using java HelloWorld. 

Hello, World!

Java Keywords Explained:

  • public: Makes the class or method accessible from anywhere. The main method must be public so the Java runtime can call it.
  • class: Defines a class, which is a blueprint for objects. All Java code resides in classes.
  • HelloWorld: The name of the class. Must match the filename HelloWorld.java.
  • static: Means the method belongs to the class itself, not an instance of the class. This allows Java to call main without creating an object.
  • void: Specifies that the method does not return any value.
  • main: The entry point method that Java looks for when running a program.
  • String[] args: An array of command-line arguments passed to the program. Each argument is a string.
  • System.out.println: Prints text to the console followed by a new line. System is a built-in class, out is the output stream, and println prints the message.
