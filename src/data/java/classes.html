A class is a blueprint for creating objects. It defines the properties (fields) and behaviors (methods) that objects will have.

Basic Class Structure:

public class Car {
    // Fields (attributes)
    String brand;
    String model;
    int year;
    
    // Method
    public void displayInfo() {
        System.out.println(brand + " " + model + " (" + year + ")");
    }
}

Creating a Class with Multiple Objects:

public class Main {
    public static void main(String[] args) {
        Car car1 = new Car();
        car1.brand = "Honda";
        car1.model = "Civic";
        car1.year = 2022;
        
        Car car2 = new Car();
        car2.brand = "Ford";
        car2.model = "Mustang";
        car2.year = 2024;
        
        car1.displayInfo(); // Output: Honda Civic (2022)
        car2.displayInfo(); // Output: Ford Mustang (2024)
    }
}

Access Modifiers:

  • public: Accessible from anywhere
  • private: Accessible only within the class
  • protected: Accessible within package and subclasses
  • default (no modifier): Accessible within package
