A class is a blueprint for creating objects. It defines the properties (fields) and behaviors (methods) that objects will have.

Basic Class Structure:

public class AIModel {
    // Fields (attributes)
    String name;
    String provider;
    int releaseYear;
    
    // Method
    public void displayInfo() {
        System.out.println(name + " by " + provider + " (" + releaseYear + ")");
    }
}

Creating a Class with Multiple Objects:

public class Main {
    public static void main(String[] args) {
        AIModel model1 = new AIModel();
        model1.name = "Claude Sonnet 4.6";
        model1.provider = "Anthropic";
        model1.releaseYear = 2025;
        
        AIModel model2 = new AIModel();
        model2.name = "GPT-4o";
        model2.provider = "OpenAI";
        model2.releaseYear = 2024;
        
        model1.displayInfo(); // Output: Claude Sonnet 4.6 by Anthropic (2025)
        model2.displayInfo(); // Output: GPT-4o by OpenAI (2024)
    }
}

Access Modifiers:

  • public: Accessible from anywhere
  • private: Accessible only within the class
  • protected: Accessible within package and subclasses
  • default (no modifier): Accessible within package
