Constructors are special methods called when an object is created. They initialize object state.

Default Constructor

public class AIModel {
    String name;
    String provider;
    int releaseYear;

    public AIModel() {
        name = "Unknown";
        provider = "Unknown";
        releaseYear = 0;
    }
}

Calling:

AIModel model = new AIModel();

Result:

name = "Unknown"
provider = "Unknown"
releaseYear = 0
  • No arguments passed
  • Values are hardcoded defaults

Parameterized Constructor

public class AIModel {
    String name;
    String provider;
    int releaseYear;

    public AIModel(String name, String provider, int releaseYear) {
        this.name = name;
        this.provider = provider;
        this.releaseYear = releaseYear;
    }
}

Calling:

AIModel model = new AIModel("Claude Sonnet 4.6", "Anthropic", 2025);

Result:

name = "Claude Sonnet 4.6"
provider = "Anthropic"
releaseYear = 2025
  • Arguments are passed
  • Values come from user input
