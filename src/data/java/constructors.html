Constructors are special methods called when an object is created. They initialize object state.

Default Constructor:

public class Car {
    String brand;
    String model;
    
    // Default constructor
    public Car() {
        brand = "Unknown";
        model = "Unknown";
    }
}

Parameterized Constructor:

public class Car {
    String brand;
    String model;
    int year;
    
    // Parameterized constructor
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
    }
    
    public void displayInfo() {
        System.out.println(brand + " " + model + " (" + year + ")");
    }
}

Using Constructors:

public class Main {
    public static void main(String[] args) {
        // Using parameterized constructor
        Car car1 = new Car("Tesla", "Model 3", 2023);
        car1.displayInfo(); // Output: Tesla Model 3 (2023)
        
        // Using default constructor
        Car car2 = new Car();
        car2.displayInfo(); // Output: Unknown Unknown
    }
}

Constructor Overloading:

public class Car {
    String brand;
    String model;
    int year;
    
    // Constructor with no parameters
    public Car() {
        this("Unknown", "Unknown", 0);
    }
    
    // Constructor with brand only
    public Car(String brand) {
        this(brand, "Unknown", 0);
    }
    
    // Constructor with all parameters
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
    }
}

Key Points:

  • Constructor name must match class name
  • No return type (not even void)
  • Called automatically when object is created
  • Can be overloaded
  • this keyword refers to current object
