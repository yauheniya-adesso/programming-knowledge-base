Type Description Example
byte 8-bit integer, stores -128 to 127. byte b = 100;
short 16-bit integer, -32,768 to 32,767. short s = 30000;
int 32-bit integer, most commonly used. int age = 25;
long 64-bit integer, for very large numbers. long distance = 100000L;
float 32-bit decimal numbers, less precise than double. float pi = 3.14f;
double 64-bit decimal numbers, more precise. double price = 19.99;
char Single 16-bit Unicode character. char grade = 'A';
boolean Represents true or false. boolean isJavaFun = true;

Reference Types

  • Store references to objects, arrays, or Strings, e.g.:
    • MyClass obj = new MyClass();
    • int[] numbers = {1,2,3};
    • String name = "Java";
