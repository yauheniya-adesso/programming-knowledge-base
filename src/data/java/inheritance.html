Inheritance allows a class to inherit properties and methods from another class, promoting code reuse.

Basic Inheritance:

// Parent class (superclass)
public class AIModel {
    protected String modelName;
    protected int version;
    
    public void train() {
        System.out.println(modelName + " is training");
    }
    
    public void deploy() {
        System.out.println(modelName + " is deployed");
    }
}

Child Class (Subclass):

// Child class extends parent
public class ChatModel extends AIModel {
    private String language;
    
    public ChatModel(String modelName, int version, String language) {
        this.modelName = modelName;
        this.version = version;
        this.language = language;
    }
    
    // Additional method specific to ChatModel
    public void generateResponse() {
        System.out.println(modelName + " is generating a response in " + language);
    }
}

Using Inheritance:

public class Main {
    public static void main(String[] args) {
        ChatModel myModel = new ChatModel("GPT", 5, "English");
        
        // Inherited methods from AIModel
        myModel.train();   // Output: GPT is training
        myModel.deploy();  // Output: GPT is deployed
        
        // ChatModel-specific method
        myModel.generateResponse(); // Output: GPT is generating a response in English
    }
}

Method Overriding:

public class VisionModel extends AIModel {
    public VisionModel(String modelName, int version) {
        this.modelName = modelName;
        this.version = version;
    }
    
    // Override parent method
    @Override
    public void train() {
        System.out.println(modelName + " is training on image data");
    }
    
    public void detectObjects() {
        System.out.println(modelName + " is detecting objects");
    }
}

Using super Keyword:

public class SpeechModel extends AIModel {
    private double accuracy;
    
    public SpeechModel(String modelName, int version, double accuracy) {
        super(); // Call parent constructor
        this.modelName = modelName;
        this.version = version;
        this.accuracy = accuracy;
    }
    
    @Override
    public void train() {
        super.train(); // Call parent method
        System.out.println("with speech recognition data");
    }
}

Key Points:

  • Use extends keyword for inheritance
  • Child class inherits all non-private members
  • @Override annotation for method overriding
  • super refers to parent class
  • Java supports single inheritance only (one parent class)
