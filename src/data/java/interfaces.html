Interfaces define a contract that classes must follow. All methods are abstract by default (until Java 8).

Defining an Interface:

public interface GenerativeModel {
    // Abstract method (implicitly public abstract)
    String generate(String prompt);
    
    // Constant (implicitly public static final)
    int MAX_TOKENS = 4096;
}

Implementing an Interface:

public class ClaudeModel implements GenerativeModel {
    private String version;
    
    public ClaudeModel(String version) {
        this.version = version;
    }
    
    @Override
    public String generate(String prompt) {
        return "Claude " + version + " response to: " + prompt;
    }
}

public class GeminiModel implements GenerativeModel {
    private String version;
    
    public GeminiModel(String version) {
        this.version = version;
    }
    
    @Override
    public String generate(String prompt) {
        return "Gemini " + version + " response to: " + prompt;
    }
}

Multiple Interface Implementation:

interface Trainable {
    void train(String dataset);
}

interface Deployable {
    void deploy(String environment);
}

public class FoundationModel implements GenerativeModel, Trainable, Deployable {
    private String name;
    
    public FoundationModel(String name) {
        this.name = name;
    }
    
    @Override
    public String generate(String prompt) {
        return name + " generating response for: " + prompt;
    }
    
    @Override
    public void train(String dataset) {
        System.out.println(name + " training on dataset: " + dataset);
    }
    
    @Override
    public void deploy(String environment) {
        System.out.println(name + " deployed to: " + environment);
    }
}

Interface with Default Methods (Java 8+):

public interface AIService {
    // Abstract method
    String sendRequest(String prompt);
    
    // Default method with implementation
    default void logUsage(String prompt) {
        System.out.println("Request logged: " + prompt.substring(0, Math.min(50, prompt.length())));
    }
    
    // Static method
    static void displayInfo() {
        System.out.println("This is an AI service interface");
    }
}

Practical Example:

interface AIProvider {
    String sendRequest(String prompt);
    void printModelInfo();
}

class AnthropicProvider implements AIProvider {
    private String modelName;
    
    public AnthropicProvider(String modelName) {
        this.modelName = modelName;
    }
    
    @Override
    public String sendRequest(String prompt) {
        System.out.println("Sending request to Anthropic API with model: " + modelName);
        return "Response from " + modelName;
    }
    
    @Override
    public void printModelInfo() {
        System.out.println("Provider: Anthropic | Model: " + modelName + " | Released: 2025");
    }
}

class MistralProvider implements AIProvider {
    private String modelName;
    
    public MistralProvider(String modelName) {
        this.modelName = modelName;
    }
    
    @Override
    public String sendRequest(String prompt) {
        System.out.println("Sending request to Mistral API with model: " + modelName);
        return "Response from " + modelName;
    }
    
    @Override
    public void printModelInfo() {
        System.out.println("Provider: Mistral | Model: " + modelName + " | Released: 2024");
    }
}

Using Interfaces:

public class Main {
    public static void main(String[] args) {
        AIProvider provider1 = new AnthropicProvider("Claude Sonnet 4.6");
        AIProvider provider2 = new MistralProvider("Mistral Large");
        
        provider1.sendRequest("Explain interfaces in Java");
        provider1.printModelInfo();
        
        System.out.println();
        
        provider2.sendRequest("What is encapsulation?");
        provider2.printModelInfo();
    }
}

Interface vs Abstract Class:

  • Interface: Use when defining a capability/contract; supports multiple inheritance
  • Abstract Class: Use when sharing code among related classes; single inheritance only
  • Interfaces cannot have instance variables (only constants)
  • All interface methods are public; abstract classes can have any access modifier
