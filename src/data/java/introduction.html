Java is a powerful, object-oriented programming (OOP) language designed for building robust, platform-independent applications.

Key Features:

  • Object-Oriented: Everything revolves around objects and classes
  • Platform Independent: Write once, run anywhere (WORA)
  • Strongly Typed: Type checking at compile time
  • Automatic Memory Management: Garbage collection handles memory
  • Rich API: Extensive standard library

Basic Program Structure:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Data Types:

  • Primitive: int, double, boolean, char, byte, short, long, float
  • Reference: Objects, Arrays, Strings
