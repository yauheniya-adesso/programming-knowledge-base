Method overloading allows multiple methods with the same name but different parameters (different number or type).

Overloading by Number of Parameters:

public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
    
    public int add(int a, int b, int c) {
        return a + b + c;
    }
}

Overloading by Type of Parameters:

public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
    
    public double add(double a, double b) {
        return a + b;
    }
    
    public String add(String a, String b) {
        return a + b;
    }
}

Example Usage:

public class Main {
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        
        System.out.println(calc.add(5, 3));           // Output: 8
        System.out.println(calc.add(5, 3, 2));        // Output: 10
        System.out.println(calc.add(5.5, 3.2));       // Output: 8.7
        System.out.println(calc.add("Hello", "World")); // Output: HelloWorld
    }
}

Key Points:

  • Method name must be the same
  • Parameters must differ in number or type
  • Return type alone is not sufficient for overloading
  • Helps improve code readability
