Methods are blocks of code that perform specific tasks. They promote code reusability and organization.

Method Syntax:

accessModifier returnType methodName(parameters) {
    // method body
    return value; // if returnType is not void
}

There are two main types of methods: Parameterized and Non-Parameterized. Create a file Calculator.java to define some methods:

public class Calculator {
    // Non-Parameterized Method:
    public void greet() {
        System.out.println("Welcome to Calculator!");
    }

    // Parameterized Methods:
    public int add(int a, int b) {
        return a + b;
    }
    
    public double multiply(double x, double y) {
        return x * y;
    }
}

To use the methods, create another file Main.java and call them: 

public class Main {
    public static void main(String[] args) {
        // Create an object of Calculator
        Calculator calc = new Calculator();
        
        // Call non-parameterized method
        calc.greet(); 
        
        // Call parameterized methods
        int sumResult = calc.add(3, 4);
        double multResult = calc.multiply(3, 4);

        // Display results
        System.out.println("Take 3 and 4");
        System.out.println(" Sum Result: " + sumResult); 
        System.out.println(" Multiply Result: " + multResult); 
    }
}

Compile all Java files in the folder and run Main: 

javac *.java
java Main
Welcome to Calculator!
Take 3 and 4
 Sum Result: 7
 Multiply Result: 12.0
