Arrays are ordered collections with zero-based indexing.

Creating Arrays:

const myArray = [];
const planets = ["Mercury", "Venus", "Earth", "Mars"];

Common Operations:

  • Access: planets[0] returns "Mercury"
  • Length: planets.length returns 4
  • Add to end: planets.push("Jupiter")
  • Remove from end: planets.pop()
  • Add to start: planets.unshift("Sun")
  • Remove from start: planets.shift()

Nested Arrays:

const matrix = [[1, 2], [3, 4], [5, 6]];
console.log(matrix[0][1]); // 2
