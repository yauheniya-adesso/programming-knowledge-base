JavaScript is a dynamically-typed language with two categories of data types:

Primitive Types (immutable):

  • String: "hello"
  • Number: 42, 3.14
  • Boolean: true, false
  • Undefined: variable declared but not assigned
  • Null: intentional absence of value
  • Symbol (ES6): unique identifier
  • BigInt (ES11): large integers

Composite Types (can hold multiple values):

  • Array: ordered collection [1, 2, 3]
  • Object: key-value pairs {name: "John", age: 30}
