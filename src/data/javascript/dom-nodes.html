DOM Hierarchy:

window
├── document
│   ├── documentElement (html)
│   │   ├── head
│   │   │   ├── meta
│   │   │   └── title
│   │   └── body
│   │       ├── form
│   │       ├── p
│   │       └── input

DOM Level 2 Node Types:

  1. Element node
  2. Attribute node
  3. Text node
  4. Comment node
  5. Document node
  6. Document Type node
  7. Document Fragment node

Accessing Form Elements:

document.forms[0].elements[0]
document.forms["formName"].elements["fieldName"]
document.formName.fieldName

Using ID (Best Practice):

const container = document.getElementById("grid_container");
