Variable names must start with a letter, underscore (_), or dollar sign ($).

Three ways to declare variables:

  • var: Function-level scope - accessible throughout the entire function
  • let: Block-level scope (ES6) - only accessible within the block {}
  • const: For constants that shouldn't be reassigned (e.g., const PI = 3.14)

Best practice: Use const by default, let when reassignment is needed, avoid var.
