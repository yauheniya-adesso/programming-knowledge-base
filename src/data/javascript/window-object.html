The window object represents the browser window and is the global object in browsers.

Window Properties:

  • window.history: Browser history
  • window.location: Current URL
  • window.navigator: Browser information
  • window.screen: Screen information
  • window.document: The DOM

Window Methods:

  • window.alert("message"): Show alert
  • window.open(url, name, features): Open new window
  • window.scrollTo(x, y): Scroll to position
  • window.onload: Execute when page loads

Example:

window.onload = function() {
  console.log("Page fully loaded");
};
